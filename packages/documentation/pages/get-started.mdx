# Get Started

1. pull the code from github repo
```
git clone https://github.com/Combo819/social-media-archiver.git
```
2. install dependencies
```
cd social-media-archiver
npm ci
```
The latest version of npm will install the dependencies for the root and all the packages. 
If you are using old version of npm, also run this command.
```
npm run bootstrap
```
